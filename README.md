# EAIV4 Mobile Web App V1.0

Mobile/PWA สำหรับรายงานสถานการณ์ภาคสนามเข้าสู่ EAIV4 ผ่าน Google Apps Script API

## ติดตั้ง GitHub Pages
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ root ของ repository `EAIV4-Mobile-WebApp`
2. Settings → Pages
3. Source: Deploy from a branch
4. Branch: main / (root)
5. Save
6. เปิด https://etechtelecom-prog.github.io/EAIV4-Mobile-WebApp/

API ถูกกำหนดไว้ใน `config.js` แล้ว
